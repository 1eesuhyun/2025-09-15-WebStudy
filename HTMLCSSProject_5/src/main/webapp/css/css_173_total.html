4. overflow

overflow:visible 바르셀로나는 주안 가르시아, 마르크 테어슈테겐, 페르민 로페스, 가비, 하피냐 등의 부상 결장이 유력하다.
overflow:hidden 바르셀로나는 주안 가르시아, 마르크 테어슈테겐, 페르민 로페스, 가비, 하피냐 등의 부상 결장이 유력하다. 적지 않은 선수들이 함께하지 못하는 가운데 PSG의 상황은 더욱 좋지 않다.
overflow:scroll 바르셀로나는 주안 가르시아, 마르크 테어슈테겐, 페르민 로페스, 가비, 하피냐 등의 부상 결장이 유력하다. 적지 않은 선수들이 함께하지 못하는 가운데 PSG의 상황은 더욱 좋지 않다.
overflow:auto 바르셀로나는 주안 가르시아, 마르크 테어슈테겐, 페르민 로페스, 가비, 하피냐 등의 부상 결장이 유력하다. 적지 않은 선수들이 함께하지 못하는 가운데 PSG의 상황은 더욱 좋지 않다.
